4. 权限管理

Oracle

授权

收回权限

对象权限可以级联收回，但是系统权限不可以级联收回。

(1) 系统权限： A 赋予一个系统权限给 B，B 再授予 C。然后A将该权限从 B 回收，此后 C 仍然有该权限。

(2) 对象权限： A 赋予一个对象权限给 B，B 再授予 C。然后A将该权限从 B 回收，此后 C 也失去了该权限。
